# rpg-desafio-apple-

- rpg säo pombos 
Este é um projeto de RPG de console, criado em Swift, que demonstra o uso de funções, tuplas e controle de fluxo.

A ideia é criar um jogo interativo onde o jogador assume o papel de um Pombo Guardião para salvar São Paulo de uma névoa cinza que drena os sonhos. O objetivo principal é aplicar os conceitos de programação Swift para simular um combate por turnos completo, com escolhas de classe (bairro: Sé, Paulista ou Ibirapuera) e habilidades especiais.

Para rodar o jogo, basta executar o arquivo main.swift dentro do Xcode como uma Command Line Tool.
A interação é feita diretamente no console (saída do Xcode). O jogo guiará você pela escolha da sua classe e, durante as batalhas, você deverá digitar o número da ação que deseja tomar:

1. Atacar
2. Usar Habilidade
3. Defender
4. Fugir

Derrote 3 inimigos aleatórios e, em seguida, enfrente o Boss final, "O Acinzentado".
